# OpenRobOps

OpenKubes Robotics Operations — GitOps-based lifecycle management for
autonomous robot fleets and industrial systems.

## make Targets

```sh
make openrobops-deploy    # deploy OpenRobOps components
make openrobops-verify    # check deployment status
make openrobops-clean     # remove OpenRobOps
```
